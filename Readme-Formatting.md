Heading
=======

We can use upto five ##### for diffrent levels of headings.  

## Sub-heading

Paragraphs are separated
by a blank line.

Two spaces at the end of a line  
produces a line break.

Text attributes _italic_, 
**bold**, `monospace`.  

For multiple  line highlighting we can use it as:  

```
#include<stdio.h>
int main() {
  printf("Mohit Mohak");
  return 0;
}
```
